# Ilon'ALA Madagascar - Plateforme E-commerce & Agrobusiness

Bienvenue sur le dépôt officiel du projet **Ilon'ALA Madagascar**. Ce projet est une plateforme web moderne et premium dédiée à la distribution de produits naturels et à la gestion de pépinières forestières.

---

## 🏗️ Structure du Projet (Méthodologie Agile)

Le projet est organisé pour faciliter la maintenance et l'évolution continue :

- **[/frontend](file:///e:/3%20-%20WEBSITE%20-%20Ilonala/frontend)** : Application client React + Vite.
- **[/backend](file:///e:/3%20-%20WEBSITE%20-%20Ilonala/backend)** : Simulation de données, schémas SQL et services métiers.
- **[/docs](file:///e:/3%20-%20WEBSITE%20-%20Ilonala/docs)** : Documentation complète, conception et cahier des charges.
- **[/agile](file:///e:/3%20-%20WEBSITE%20-%20Ilonala/agile)** : Gestion de projet, Backlog et suivi des Sprints.

## 🚀 Installation & Lancement

### 1. Cloner le projet
```sh
git clone <git-url>
cd ilon-ala
```

### 2. Installer les dépendances
```sh
npm install
```

### 3. Lancer le serveur de développement (Root)
```sh
npm run dev
```

---

## 🛠️ Stack Technique

- **Frontend** : React, Vite, Tailwind CSS, Lucide Icons, Shadcn/UI.
- **Backend (Prototype)** : Simulation CRUD via LocalStorage, SQL Schema pour migration future.
- **IA/Management** : Géré avec la méthodologie Agile Scrum pour une itération rapide.
